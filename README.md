# knot-losing-you
Autonomous UGV that follows you around the boat park using OAK-D Lite and LiDAR. No sea legs required.

### Development Machine

From the repo root:

```bash
# Change directory to the module
cd /ugv-follower

# Install the package
pip install -e ".[dev]"
```

Technical Specifications Setup:
- Raspberry Pi 5
- Waveshare UGV Rover
- OAKd Lite
